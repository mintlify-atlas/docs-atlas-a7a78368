## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="How do I install the app on iOS?">
    On iOS, the app is distributed via **AltStore** (no jailbreak required). Follow these steps:
    
    1. Install AltStore following the official guides:
       - [macOS Installation Guide](https://faq.altstore.io/altstore-classic/how-to-install-altstore-macos)
       - [Windows Installation Guide](https://faq.altstore.io/altstore-classic/how-to-install-altstore-windows)
    
    2. Once AltStore is installed, download the `.ipa` file from [GitHub Releases](https://github.com/ivansaul/cheat-sheets/releases)
    
    3. Install via AltStore Source or manual installation
    
    **Important:** Open the app from the home screen icon, not the "OPEN" button after installation, to avoid deeplink issues.
  </Accordion>

  <Accordion title="Where can I download the app?">
    The app is available on multiple platforms:
    
    - **Android:** [Google Play Store](https://play.google.com/store/apps/details?id=com.ivansaul.cheatsheets) or [GitHub Releases](https://github.com/ivansaul/cheat-sheets/releases) (.apk, .aab)
    - **iOS:** [AltStore](https://tinyurl.com/ivansaulAltstore) or [GitHub Releases](https://github.com/ivansaul/cheat-sheets/releases) (.ipa)
  </Accordion>

  <Accordion title="Is offline mode available?">
    Offline support is currently in development and planned for a future release. You can track progress on the [roadmap](https://github.com/ivansaul/cheat-sheets#road-map).
  </Accordion>

  <Accordion title="What programming languages are supported?">
    The app provides comprehensive cheat sheets for:
    - Python
    - Rust
    - Swift
    - JavaScript
    - Kotlin
    - Go
    - Git
    - And more!
  </Accordion>

  <Accordion title="How can I contribute to the project?">
    We welcome contributions! Please refer to the [Development Guide](https://github.com/ivansaul/cheat-sheets/blob/main/DEVELOPMENT.md) for setup instructions and contribution guidelines.
  </Accordion>

  <Accordion title="Is the app free?">
    Yes! Cheat Sheets is completely free and open-source under the GPLv3 license.
  </Accordion>

  <Accordion title="I found a bug. What should I do?">
    Please report bugs by opening an issue on [GitHub](https://github.com/ivansaul/cheat-sheets/issues). Include:
    - Device and OS version
    - App version
    - Steps to reproduce the issue
    - Screenshots if applicable
  </Accordion>
</AccordionGroup>
