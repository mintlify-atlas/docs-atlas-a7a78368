## Version Information

The app follows semantic versioning:

- **Major version:** Significant changes and new features
- **Minor version:** Smaller features and improvements
- **Patch version:** Bug fixes and minor updates
- **Build number:** Internal build identifier

Current format: `0.4.0+9` (version 0.4.0, build 9)

## Known Issues

<Warning>
  **iOS Launch Issue:** When installing via AltStore, opening the app via the "OPEN" button may cause it to not work correctly due to a deeplink issue. Please open it from the **home screen icon** instead. This will be fixed in a future update.
</Warning>
