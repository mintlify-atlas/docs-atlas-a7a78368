## Welcome to Cheat Sheets

**Cheat Sheets** is a comprehensive mobile reference guide designed to help developers quickly access essential information for multiple programming languages and tools. Whether you're learning a new language or need a quick refresher, this app provides concise, actionable reference material at your fingertips.

## What is Cheat Sheets?

Cheat Sheets is a cross-platform mobile application built with Flutter that provides quick reference guides for:

- **Python** - Syntax, data structures, and common operations
- **Rust** - Language fundamentals and best practices
- **Swift** - iOS development essentials
- **JavaScript** - Modern JS features and patterns
- **Kotlin** - Android development and language features
- **Go** - Concurrency, syntax, and standard library
- **Git** - Version control commands and workflows
- And many more!

## Technology Stack

Cheat Sheets is built with modern technologies:

- **Flutter** - Cross-platform UI framework
- **Dart** - Programming language (SDK >= 3.1.2)
- **Riverpod** - State management
- **Go Router** - Navigation
- **Dio** - HTTP client
- **Markdown Widget** - Content rendering
